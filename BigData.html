Description

The motivation behind the project is that as a self-taught guitar player myself, I realized that it is often difficult for beginners with no music background to memorize the connection between fretboard positions and note pitches. Besides, finger arrangements and correct rhythms can also be hard to remember. So, i developed this project in the hope that the following improvements in learning can be achieved:
1. Guitar notes will be visualized through HUD
2. Metronome can be set directly using hand gestures.
In the future, if time allowed, the following developments will be made:
1. Instead of notes in English letter, music sheet will appear.
2. Finger movements of proficient players can be recorded and revisualized so that beginners can learn songs more quickly.
